Генератор ссылок на оплату ChatGPT (Старый Stripe)

Перетащите кнопку ниже на панель закладок:

📌 ChatGPT Payment

Как пользоваться

  1. Перетащите зелёную кнопку выше на панель закладок браузера
  2. Откройте chatgpt.com и войдите в нужный аккаунт
  3. Кликните по закладке, находясь на chatgpt.com
  4. В появившемся окне выберите тариф (1 = Plus, 2 = Go)
  5. Скопируйте ссылку оплаты из следующего окна и пришлите мне
⚠️ Не переходите по скопированной ссылке — это может сломать оплату.
Примечание: тарифы Pro x5 и Pro x20 не поддерживаются у меня — OpenAI использует встроенный Stripe Checkout для этих тарифов, и внешнюю ссылку получить невозможно.
